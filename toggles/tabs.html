Tabs #

Default Tabs #

The tab button is used wherever the user needs to choose between different content types or make a visual selecten between a different radio values.

Preview #


Markup #

<div class="crowny__toggles__tabs">
    <label class="tab" for="radio-1">
        <input id="radio-1" type="radio" v-model="activeRadio" value="0" name="radio-group" checked>
        <i class="icon-QR"></i> 
        <span>QR</span>             
    </label>          

    <label class="tab" for="radio-2">
        <input id="radio-2" type="radio" v-model="activeRadio" value="100" name="radio-group">            
        <span>No-icon</span>            
    </label>           

    <label class="tab" for="radio-3">
        <input id="radio-3" type="radio" v-model="activeRadio" value="200" name="radio-group">
        <i class="icon-Crown"></i> 
        <span>Crown</span>            
    </label>      

    <div class="crowny__glider"></div>
</div>

Glider #

Important

The purple glider/slider does not work out of the box. This has to be done programatically, see the basic example in Vue 3 below.

Adjust to your needs accordingly ðŸ˜ƒ

<div class="crowny__toggles__tabs">        
    <label class="tab" for="radio-1">
        <input id="radio-1" type="radio" v-model="activeRadio" value="0" name="radio-group" checked>
        <span>Value 1</span>             
    </label>          

    <label class="tab" for="radio-2">
        <input id="radio-2" type="radio" v-model="activeRadio" value="100" name="radio-group">            
        <span>Value 2</span>            
    </label>           

    <div class="crowny__glider" :style="`transform: translateX(${activeRadio}%)`"></div>        
</div> 

  setup: () => {
    const state = reactive({
      activeRadio: 0
    })

    return {
      ...toRefs(state)
    }
  }
