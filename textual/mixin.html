Typography #

Mixin #

All typography coming from the Crowny-UI can be used styled via the typography mixin:
You can call the SCSS mixin on any element to give it it's textual styling.

@include mixins.typography('body', 'base');

/*
font-family: 'Inter', sans-serif;
font-weight: 400;
line-height: 1.5;
font-size: 1rem; (16px)
*/

Important

mixins. namespacing is required.

TIP

When calling the @include mixins.typography() we use 2 parameters.

  1. Key (Name of the array)
  2. Entry (Name of the option)

Titles/Headings/Headlines #

Within the typography mixin we have an array of headlines.

Example

@include mixins.typography('headlines', 'h1');
KeyEntryFont size
headlinesh124px - 64px
headlinesh224px - 30px
headlinesh320px - 30px
headlinesh424px - 30px
headlinesh524px - 30px

Body #

Within the typography mixin we have an array of body styles. Body always uses the font Inter

Example

@include mixins.typography('body', 'xl');
KeyEntryFont sizeFont weight
bodyxs12px400
bodysm14px400
bodybase16px400
bodylg18px400
bodyxl20px400

Utils #

Within the typography mixin we have an array of utility styles. Utils always uses the font Inter

Example

@include mixins.typography('utils', 'caption');
KeyEntryFont sizeFont weightUppercase
utilslink16px500No
utilscaption16px600No
utilsbreadcrumbs14px500No
utilslabel12px500Yes
utilsmicro9px500No

Buttons #

Within the typography mixin we have an array of button styles. Buttons always uses the font Inter

Example

@include mixins.typography('buttons', 'xs-small');
KeyEntryFont sizeFont weightUppercase
buttonsxs-small12px500No
buttonsmedium-large14px500No
buttonsxl-xxl16px500No
