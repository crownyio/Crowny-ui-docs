Statistic Card #

The statistic card is a horizontal, but narrow card that shows the most basic statistics

Markup #

Important

The statistic card component comes without any margins, thus if you are create for example; A list of statistic card, you need to add the spacing in between the components yourself.

Used Atoms

  1. Icon Holder
<div class="crowny__statistic__card">
    <!-- icon holder component -->
    <div class="crowny__icon__holder">
        <i class="icon-Trophy"></i>
    </div>

    <div class="crowny__statistic__card__content">
        <span class="title">Reward</span>
        <span class="description">All of the rewards?</span>
    </div>

    <div class="crowny__statistic__card__stats">
        <span class="statistic color-accents-emerald">+300</span>
    </div>
    i
</div>
