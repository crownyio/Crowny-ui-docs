Buttons #

Default #

Crowny-UI has a wide variety of buttons at your disposal. Buttons get their default styling from the global class crowny__btn.

TIP

Button can be used as an anchor (link) or button element.

Preview #

Default

Markup #

<a href="#" class="crowny__btn"></a>
<button class="crowny__btn"></button>

Disabled #

Adding a disabled state/and/or class to a button will result in the button being not clickable.

WARNING

A disabled button will always show the same styling/colors no matter what Color Class you apply.

Preview #

Disabled

Markup #

<a href="#" class="crowny__btn disabled">Disabled</a>
<button class="crowny__btn" disabled>Disabled</button>

Colors #

Additionally we provide you with classes to define the buttons call-to-action purpose, this will give the button it's appropriate color.

Important

Button color options should be added to the default button class class="crowny__btn"

Preview #

ColorsColorsColorsColorsColorsColorsColorsColorsColors
Color classDescriptionUsage
primaryPrimary button is used a our main call-to-actionclass="primary"
secondarySecondary button is used next to the primary button, used as secondary call-to-actionclass="secondary"
tertiaryTertiary button is used when primary, secondary is already being usedclass="tertiary"

outlineOutline is often used as a replacement for the secondary button in cards e.gclass="outline"
buyBuy button is used as a 'positive' call-to-actionclass="buy"

Markup #

<a href="#" class="crowny__btn {{ ColorClass }}">Default</a>
<button class="crowny__btn {{ ColorClass }}">Default</button>

Outline #

Additionally we provide you with classes to define the buttons outline variant.

Important

Button outline options should be added to the default outline class class="outline__"

Preview #

Darkmode

When using outline buttons on a dark background we provide you a darkmode class. This class can be added to the button e.g:

<a href="#" class="crowny__btn outline__primary darkmode">Darkmode</a>

<button class="crowny__btn outline__{{ ColorClass }}">Outline</button>

Sizing #

We also provide you with pre-defined sizing options to give the button it's needed padding and size.

Important

Button sizing options should be added to the default button class class="crowny__btn"

Preview #

Icons #

Last but not least, we provide you with the option to add icons to any of the buttons mentioned above. Adding icons can be done in a variety of ways.

Important

For buttons that need an icon, you need to add the class class="with__icon"

Preview #

Right Left
Icon optionsDescriptionUsage
icon__rightMakes room for an icon at the right side of the buttonclass="icon__right"
icon__leftMakes room for an icon at the left side of the buttonclass="icon__left"

Markup #

<a href="#" class="crowny__btn with__icon"><i class="icon-Crown"></i></a>
<a href="#" class="crowny__btn with__icon {{ iconOptions }}">Right <i class="icon-ChevronRight"></i></a>
<a href="#" class="crowny__btn with__icon {{ iconOpions }}"><i class="icon-ChevronLeft"></i> Left</a>

Icons Rounded #

In some cases, for example our notification overview we would want to have icons that behave as buttons. By default only the icons are visible, upon hover it will show a a button style (rounded).

Preview (hover) #

Markup #

<a href="#" class="crowny__btn with__icon rounded"><i class="icon-Crown"></i></a>
<a href="#" class="crowny__btn with__icon rounded">Right <i class="icon-ChevronRight"></i></a>
<a href="#" class="crowny__btn with__icon rounded"><i class="icon-ChevronLeft"></i> Left</a>
