Curves #

Curves (radius) #

Curves are pre-defined from within the design system, Crowny-UI gives you an array of curves to use with border-radius.

Important

Currently there's no Curve classes available. vars. namespacing is required.

OptionPX Amount

Usage #

.curve__example {
    border-radius: vars.$curve-8;
}
