Crowny gradients #

Gradients #

All gradients coming from the Crowny-UI can be used in two different ways:
You can apply a class (specified per gradient below) to any element that needs a gradient background e.g

<div class="gradient-primary-1"><div>

Or use the SCSS mixin instead (and use the gradient name specified below)

@include mixins.gradient('primary-1', 'linear');
@include mixins.gradient('primary-1', 'radial');

Important

mixins. namespacing is required.

Important

Currently we have no radial gradients defined in the Crowny-UI.
The example above is purely an example of the gradient mixin.
